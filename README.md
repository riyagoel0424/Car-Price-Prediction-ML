# Car Price Prediction Using Machine Learning

## Objective
Predict the selling price of used cars using machine learning algorithms.

## Dataset Features
- Year
- Present Price
- Kms Driven
- Fuel Type
- Seller Type
- Transmission
- Owner

## Models Used
1. Linear Regression
2. Lasso Regression
3. Random Forest Regressor

## Evaluation Metric
R² Score

## Results
- Linear Regression R² Score ≈ 0.87
- Random Forest R² Score ≈ 0.98

Random Forest performed better than Linear Regression.
